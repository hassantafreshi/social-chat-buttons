# WP Social Chat Button

A lightweight, modern WordPress plugin to manage your social support contact methods (WhatsApp, Telegram, Signal, etc.) with a clean admin panel and an optional floating button on the frontend.

## Features
- Admin menu with Panel and Settings pages
- Add unlimited contact methods via a responsive modal (Save/Cancel)
- Auto-detects input type (Username/Phone) based on selected network
- Secure saving with nonces, capability checks, and sanitization
- i18n-ready (text domain: `wp-social-chat-button`)

## Requirements
- WordPress 4.7+
- PHP 5.4+

## Installation
1. Upload the plugin folder to `wp-content/plugins` or install via the Plugins screen.
2. Activate the plugin.
3. Go to “WP Social Chat Button → Panel” and add your contact methods.

## Development
This plugin follows WordPress coding standards and aims to be fast, secure, and user-friendly.
