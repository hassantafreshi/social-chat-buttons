== Description ==
WP Social Chat Button یک افزونه ساده و مدرن برای مدیریت راه‌های ارتباطی شبکه‌های اجتماعی در پیشخوان وردپرس است. با این افزونه می‌توانید از طریق یک صفحه «Panel» در منوی «WP Social Chat Button»، هر تعداد راه ارتباطی (مانند WhatsApp، Telegram، Signal و شبکه‌های دیگر) اضافه کنید. بر اساس انتخاب شبکه، نوع ورودی به صورت خودکار (نام کاربری یا شماره تماس) تغییر می‌کند. رابط کاربری افزونه واکنش‌گرا، موبایل‌فرندلی و کاربرپسند است.

ویژگی‌ها:
- منوی اختصاصی در پیشخوان: WP Social Chat Button با زیرمنوهای Panel و Settings
- افزودن بی‌نهایت راه ارتباطی با UI مدرن و دکمه‌های ذخیره/کنسل در مودال
- تشخیص خودکار نوع ورودی (Username/Phone) بر اساس شبکه
- ذخیره‌سازی امن با استفاده از Nonce و Sanitization
- رعایت استانداردهای تیم توسعه وردپرس (امنیت، سرعت، i18n)

== Installation ==
1. پوشه افزونه را در مسیر wp-content/plugins آپلود کنید یا از بخش افزونه‌ها نصب کنید.
2. افزونه را فعال کنید.
3. از منوی «WP Social Chat Button» وارد «Panel» شوید و راه‌های ارتباطی خود را اضافه کنید.
